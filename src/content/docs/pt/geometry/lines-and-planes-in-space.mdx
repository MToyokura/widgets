## Determinação de um plano

Um plano é determinado quando satisfaz qualquer uma das seguintes condições:

- Três pontos não colineares
- Uma reta e um ponto fora dela
- Duas retas secantes

## Uma reta perpendicular a um plano

Para afirmar que uma reta é perpendicular a um plano, basta mostrar que ela é perpendicular a duas retas secantes nesse plano. Uma vez satisfeita essa condição, a reta é definida como perpendicular a todo o plano e, como resultado, o ângulo que ela forma com qualquer reta existente no plano será de 90 graus.
