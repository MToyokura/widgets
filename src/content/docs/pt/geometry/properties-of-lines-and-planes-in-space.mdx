## Reta perpendicular a um plano

Para afirmar que uma reta é perpendicular a um plano, basta mostrar que ela é perpendicular a duas retas concorrentes contidas nesse plano. Quando essa condição é satisfeita, a reta é perpendicular a todo o plano e, consequentemente, forma um ângulo de 90° com qualquer reta do plano.

## Retas paralelas a uma terceira reta

Se duas retas distintas são paralelas a uma mesma terceira reta, então elas são paralelas entre si.

## Retas perpendiculares a uma terceira reta

Diferentemente do que ocorre na geometria plana, no espaço, duas retas perpendiculares a uma mesma terceira reta não são necessariamente paralelas entre si.
